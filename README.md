# MSA_structure

작가는 웹소설을 편당 금액과 함께 등록할 수 있으며
독자는 돈을 지불하여 이를 감상할 수 있다. <br>
관련하여 기본적인 도서에 대한 관리가 필요하고, 결제를 위한 내용이 필요하다 by monolithic & msa structure <br>

독자는 소설을 전부 즉기 결제로만 이용가능하다.(캐시 기능 x)
